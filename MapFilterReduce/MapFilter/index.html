Map + Filter

Ainda a titulo de curiosidade e demonstração de flexibilidade das funções, podemos utilizar o map e o filter juntos, para atingir um objetivo.
Neste exemplo, vamos iterar sobre um array de numeros, filtrar os multiplos de 2 e multiplica-los por 2.

const numeros = [2,3,6,5,10];
const arrayMapFilter = numeros
.filter(numero => numero % 2 === 0)
.map(numero => numero * 2);

Aqui, como mencionado, estamos iterando sob um array.
A partir disso, filtramos os numeros que são multiplos de 2, atraves da função filter.
Os numeros que atenderam a esta validação, são então passados a ao map, que então itera sob eles, e realiza a sua ação, multiplicando os valores.
Como o map é a ultima função chamada, o retorno é um novo array, contendo apenas os numeros filtrados pela ação do filter e posteriosmente multiplicados pela ação do map
