Map

Map é uma função, de iteração em arrays, que espera uma função de callback, função esta que pode ser utilizada para manipular o array onde ela esta iterando.
Neste exemplo, vamos iterar sobre um array de numero, e multiplicar por 2.

Primeiro, um exemplo de como seria, do modo tradicional com um for

const numeros = [2,3,6,5,10];
let novoArrayNumeros = [];
for(let i = 0; i < numeros.length; i++){
novoArrayNumeros.push(numeros[i]*2);
}

Acima temos a iteração comum, com um laço for.
Podemos notar que, para atingir nosso objetivo, foi necessario criar a variavel novoArrayNumeros fora do escopo de execução do for, para que assim fosse possivel, iterar, calcular e criar uma nova saída.

Agora, um exemplo com o map

const novoArrayNumerosMap = numeros.map(function(numero){
return numero * 2;
});

Acima, tem a mesma iteração, porem utilizando o map. Como a função map pede uma função de callback como parametro, e sempre ira retornar um array, nós realizamos a transformação do valor, na função de callback, e retornamos os novos valores, para o novo array.
Deste forma, ambos arrays coexistem, e nenhuma variavel foi criada fora do escopo para atender a esse cenario.
Outro detalhe, é o fato de que utilizando o map para atender a este cenario, nosso desenvolvimento ficou mais legivel e limpo.

Porem, existem ainda outras formas de deixar esta implementação ainda mais compacta, se utilizarmos outros recursos qud o Javascript fornece.

const novoArrayArrow = numeros.map(numero => numero * 2);

Acima, temos a mesma implementação, utilizando alem do map, uma arrow function.
Desse modo, conseguimos realizar a mesma iteração, em apenas uma linha.
